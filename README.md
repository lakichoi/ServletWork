# ServletWork
## A. 개요
Servlet/JSP를 활용하여 MVC2모델의 기본적인 CRUD와 로그인,로그아웃 구현.

## B. 구현목록

1.로그인   
2.로그아웃   
3.회원목록   
4.회원가입   
5.회원수정   
6.회원삭제
